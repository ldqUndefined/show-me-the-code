# 几种图片格式的优缺点

| 图片类型 | 优点                                        | 缺点                     | 应用场景                                       |
| -------- | ------------------------------------------- | ------------------------ | ---------------------------------------------- |
| PNG      | 无损，支持透明                              | 画质中等                 | 应用于大多数中小图且画质较好的需求             |
| SVG      | 矢量图，伸缩不变形，用XML描述图片，易于修改 | 编写麻烦，性能差         | 多用于icon                                     |
| GIF      | 无损，支持动画和透明                        | 只支持8bit索引色，画质差 | 动画需求                                       |
| JPEG     | 文件小                                      | 有损压缩，画质损失       | 不考虑特别好的画质且需要响应速度快，如大背景图 |
